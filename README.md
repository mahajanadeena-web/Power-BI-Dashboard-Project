# Power-BI-Dashboard-Project

Title: Data Professional Survey Breakdown

Subtitle: Power BI Dashboard Project

Project Summary
This project analyses survey responses from 630 global data professionals. Using Power BI, I explored trends in job roles, salaries, programming preferences, and career satisfaction. The dashboard offers insights into the challenges and rewards of working in data.

Dataset Overview
•	Source: Anonymous survey data
•	Size: 630 responses
•	Key fields: Job title, salary, country, age, programming language, happiness metrics, career switch status

Data Cleaning & Transformation Steps
•	Removed irrelevant columns (e.g., browser, referrer)
•	Split columns by delimiter and digit/non-digit (e.g., salary ranges)
•	Duplicated columns for derived metrics
•	Replaced inconsistent values (e.g., standardizing country names)
•	Created custom columns (e.g., salary midpoint, age groupings)
